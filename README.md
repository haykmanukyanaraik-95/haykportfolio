# Hayk Manukyan — Portfolio

Personal portfolio website of Hayk Manukyan — UX / Product Designer.

Single-page site built with Next.js, Tailwind CSS, ReactBits animated components, and Three.js WebGL background.

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Tech Stack

- Next.js 16 (App Router) + React 19
- Tailwind CSS v4
- motion/react (animations)
- Three.js + postprocessing (PixelBlast WebGL background)
- GSAP + ScrollTrigger (scroll-triggered reveal)
- shadcn/ui (base components)
- Roboto (Google Fonts)
- Flaticon uicons (icons)

See `CLAUDE.md` for full project documentation.
